# Remotion Anti-PPT Example

A minimal Remotion project demonstrating the anti-PPT component library.

## Scripts

```bash
npm install
npm run typecheck   # tsc --noEmit
npm run start       # Remotion studio
npm run build       # Render the demo
```

## Composition

- `AntiPPTDemo` at 1920x1080, 30fps.
- Uses `CameraRig` to drive five scenes with continuous Z-axis push transitions.
- Demonstrates `KineticText`, `GlowCard3D`, `GlowSurface`, `TypewriterText`, `Cursor`, `GrainOverlay`, `Vignette`, `AmbientGlow`, and `SceneTransition`.
